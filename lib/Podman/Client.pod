=head1 Podman::Client

API connection client.

 my $Client = Podman::Client->new(
     Connection => 'http+unix:///run/user/1000/podman/podman.sock',
     Timeout       => 1800,
 );
 
 my $Response = $Client->Get(
     'version',
     Parameters => {},
     Headers    => {},
 );

=head1 Attributes

=head2 C<has 'Connection'; #43>

API connection Url. Possible connections are via UNIX socket (default)
or tcp connection.

 * http+unix
 * http
 * https

=head2 C<has 'Timeout'; #51>

API connection timeout, default C<3600 seconds>.

=head2 C<has 'UserAgent'; #59>

API connection object.

=head2 C<has 'RequestBase'; #68>

API request Url depends on connection Url and Podman service version.

=head1 Methods

=head2 C<sub Get($Self, $Path, %Options); #157>

Send API get request to path with optional parameters and headers.

=head2 C<sub Post($Self, $Path, %Options); #171>

Send API post request to path with optional parameters, headers and
data.

=head2 C<sub Delete($Self, $Path, %Options); #198>

Send API delete request to path with optional parameters and headers.

=cut
